# Flextock-task

### Tools and Languages
- Python
- requests library (to call api)
- psycopg2 (to connect to postgres database)


### How to run the application

- Edit database configuration in database.py file
```sh
    pip install -r requirements.txt
```
- To run the application 
```sh
    python main.py
```
- To run unit tests

```sh
    python tests.py
```
